# Rules Index

`docs/rules/`는 에이전트가 준수해야 하는 **기계적 운영 규칙** 저장소다.

## 파일 안내
- `agent.md`: 역할, 실행 루프, 실행계획, 품질 게이트
- `architecture.md`: 레이어 구조, 의존 방향, 위반 처리
- `conventions.md`: 코딩/커밋/환경/금지 패턴
- `cleanup.md`: 엔트로피 관리, 정리 기준, 드리프트 처리
- `frontend.md`: UI 변경 검증 및 증빙
- `observability.md`: 로그/메트릭/헬스/추적 운영 검증

## 운영 원칙
1. 규칙은 짧고 명확하게 유지
2. 추상 원칙보다 검증 가능한 실행 절차 우선
3. 코드와 문서가 어긋나면 문서도 버그로 처리
